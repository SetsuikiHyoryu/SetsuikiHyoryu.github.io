使用hexo寫博文的流程

以下命令輸入在命令行終端中

一、安裝Node.js

  1. 進入https://nodejs.org/下載推薦版

    • 除了修改路徑以外全部默認
    • 由Node包和npw包組成

  2. 在命令行終端中使用查看版本命令確認是否安裝成功

    • node -v
    • npm -v

二、使用hexo框架創建博客

  1. 安裝hexo

    • npm install -g hexo-cli
    • 如連不上外網需設置加速鏡像站詳見參考資料1

  2. 驗證是否成功安裝

    • hexo -v：查看版本

  3. 生成一個供存放博客框架的文件夾

    • mkdir 文件夾名
    • 對建立的博客有任何不滿都可以删除這個文件來重來

  4. 建立一個框架（網站）

    • hexo init

  5. 建立一個本地服務（打開hexo）

    • hexo server
    • 僅作預覽用
    • 需要使用Ctrl+C關閉

  6. 新建博文

    • hexo new "文件名"

  7. 清理緩存

    • hexo clean

  8. 生成文件

    • hexo generate

三、使用GitHub作爲博客的倉庫

  1. 在GitHub中新建倉庫

    • 倉庫名必須爲“GitHub用戸名.github.io”

  2. 安裝hexo的git插件

    • npm install --save hexo-deployer-git

  3. 修改文件

    • _config.yml
    • 在博客的本地根目録下
    • ## Docs: https://hexo.io/docs/one-command-deployment
deploy:
type: 'git'
repo: 'https://github.com/SetsuikiHyoryu/SetsuikiHyoryu.github.io.git'
branch: 'master'```

4.部屬到遠端
- hexo deploy
- 過程中需要登入GitHub帳號
- 如果想反映修改過的内容必須每次調用clean, generate, deploy指令
